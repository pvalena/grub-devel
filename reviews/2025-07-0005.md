# AI Review: MR !52 - Add TPCM module support

Adds TPCM (Trusted Platform Control Module) verifier for measured boot, similar to TPM but with Chinese national standard.

- **Missing header file**: grub-core/commands/tpcm.c:24 includes `<grub/efi/tpcm.h>` but file is not added to commit. Causes undefined `TPCM_MAX_BUF_SIZE` and `grub_tpcm_measure_memory()`. Won't compile.

- **Wrong type mask** (grub-core/commands/tpcm.c:72): Uses `(type & GRUB_FILE_TYPE_MASK)` but `type` is `enum grub_verify_string_type`, not `enum grub_file_type`. Masking operation is meaningless.

- **Buffer overflow risk** (grub-core/commands/tpcm.c:68-75): Allocates `strlen(str) + strlen(prefix) + 5` but `grub_snprintf(context, 4, "%d|", type)` truncates type numbers >= 100 to 3 digits. If verify string types can be >= 100, this causes silent data corruption.
