# AI Review: MR !138 - Scan all BARs on PCI serial devices for I/O port

1 commit fixing PCI serial device detection to check all 6 BARs
instead of only BAR0.

**Commits:**
1. **5896e1f28** - term/serial: scan all BARs on PCI serial devices for I/O port

## Issues Found

No issues found.

The old code read only BAR0 and skipped any device whose BAR0 was not
I/O space. The new code iterates BARs 0-5, correctly skipping the
upper 32-bit slot of 64-bit memory BARs (`GRUB_PCI_ADDR_MEM_TYPE_64`),
and takes the first I/O BAR found. The `bar == 0` sentinel after the
loop is safe because any valid I/O BAR has bit 0 set
(`GRUB_PCI_ADDR_SPACE_IO = 0x01`), so it can never be zero. The rest
of the function (`bar & GRUB_PCI_ADDR_IO_MASK`, port allocation, error
paths) is unchanged and correct.
