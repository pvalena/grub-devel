# AI Review: MR !47 - Option to set title of submenus

Adds `--menutitle` option to menuentry/submenu commands to customize submenu titles.

- **Critical: Double increment in loop** (grub-core/commands/menuentry.c:268): Loop has both `for` increment (`t++`) and body increment (`*t++`), skipping every other character in length calculation. Causes wrong buffer size allocation. Should be `for (t = title; *t;)` (remove loop increment) or `len += (*t == '\'' ? 4 : 1)` (remove body increment), matching setparams_prefix pattern at line 236-237.
