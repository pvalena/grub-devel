# AI Review: MR !68 - Add XBOOTLDR partition support

2 commits: Adds support for Extended Boot Loader Partition (XBOOTLDR) to store BLS entries/UKIs when ESP is full.

- **Double-free risk** (grub-core/commands/blsuki.c:1380,1407 in commit 1): Code calls `grub_free(esp_disk->partition)` then later `grub_disk_close(esp_disk)`. The close function will attempt to free the partition again, causing double-free. Should use `grub_partition_t saved = esp_disk->partition; esp_disk->partition = NULL;` without freeing, then restore before closing.
