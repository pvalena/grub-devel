## Issues Found

### Issue 1: Missing `format` attribute on `grub_user_error` declaration

**Location:** `include/grub/err.h`, line ~93

The `grub_user_error` declaration is missing the `GNU_PRINTF` format
attribute. `grub_error` has it (`__attribute__ ((format (GNU_PRINTF, 5,
6)))`), but `grub_user_error` does not. Without it, the compiler will
not warn about format string / argument mismatches in future callers.

```diff
-grub_err_t EXPORT_FUNC(grub_user_error) (grub_err_t n, const char *fmt, ...);
+grub_err_t EXPORT_FUNC(grub_user_error) (grub_err_t n, const char *fmt, ...)
+	__attribute__ ((format (GNU_PRINTF, 2, 3)));
```
